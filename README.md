# Introduction to Computer Programming (Python)

## About the Project
This project evaluates the understanding and practical application of fundamental Python programming concepts.
It covers core topics such as data types, control flow, functions, object-oriented programming, and exception handling through hands-on coding problems.

## Skills Covered
- Python programming fundamentals  
- Data types, indexing, and slicing  
- Operators, in-built functions, and methods  
- Conditional statements and indentation  
- Loops and iterations (including advanced looping)  
- Custom function creation  
- Object-Oriented Programming (OOPs)  
- Exception handling  

## Deliverables
- Python coding solutions for all given questions  
- Google Colab link containing the complete code  
- Video explanation for any five selected coding questions  

## Evaluation Criteria (Rubrics)
### Coding Assessment
- Correctness of solutions  
- Efficient use of Python constructs  
- Clean and well-structured code  

### Colab Submission
- Properly organized Python code  
- Correct and accessible Colab link  

### Video Explanation
- Clear explanation of logic and approach  
- Coverage of key programming concepts  
- Structured and easy-to-understand presentation  

## Outcome
This project demonstrates strong foundational knowledge of Python programming, logical problem-solving ability, and clear explanation skills required for real-world software and data-related roles.
